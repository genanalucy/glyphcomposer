## 论文当前已锁定的规则

### 章节规则

- 当前主要写作对象是开题报告正文。
- 第二部分是：`选题依据`
- 第三部分是：`研究内容、研究目标及拟解决的关键问题`
- 第四部分是：`研究方案及可行性分析`
- 第五部分是：`论文选题的特色与创新之处`
- 第六部分是：`研究计划及预期研究结果`
- 第七部分“研究工作基础”此前明确**本轮不写**，除非用户重新要求。

### 引文规则

- **只有第二部分插入正文文献标注**。
- 第三、四、五、六部分**不加正文引文**。
- 当前第二部分引文已经按**首次出现顺序**从 `[1]` 到 `[36]` 连续重排。
- 文末参考文献列表已与第二部分的首次出现顺序同步。

### 写作风格规则

- 用户非常在意“AI 味”和模板腔。
- 不要写成“提纲扩写稿”。
- 不要大量使用以下表达：
  - `其一 / 其二 / 其三`
  - `第一 / 第二 / 第三`
  - `在数据层面 / 在模型层面 / 在扩展层面`
  - `从数据可行性看 / 从工程可行性看`
  - `本课题的特色首先体现在……`
  - `第二个特色体现在……`
  - `第三个特色体现在……`
  - `起点是……`
  - `基线模型研究会在这一基础上展开`
  - `研究的收口环节放在……`
- 可以使用学术连接词，但要控制频率，不要连续多段重复：
  - `一方面……另一方面……`
  - `不是……而是……`
  - `这意味着……`
  - `换言之……`
  - `综合来看……`
