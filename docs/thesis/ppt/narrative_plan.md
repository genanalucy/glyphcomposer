# 开题答辩 PPT 叙事草案

## 目标

围绕“基于部件与结构条件约束的方块字字形生成方法研究”制作一套 12 页开题答辩 PPT。整套汇报采用“问题提出 -> 研究空缺 -> 任务定义 -> 方案设计 -> 可行性与计划”的逻辑，不按开题报告书表单栏目逐项照搬。

## 受众

- 开题评审老师
- 指导教师
- 具备人工智能、模式识别或计算机视觉背景的研究生答辩场景

## 页面结构

1. 封面
2. 目录
3. 研究背景与问题提出
4. 研究意义与应用场景
5. 国内外研究现状与研究空缺
6. 任务定义与研究目标
7. 总体研究方案
8. 核心方法设计
9. 拟解决的关键问题
10. 创新点与可行性分析
11. 研究计划与预期成果
12. 致谢与 Q&A

## 备注策略

- 备注不是提纲，而是逐页可直接照着讲的讲稿。
- 每页讲稿控制在自然口语节奏下约 35 到 70 秒。
- 讲稿语气尽量接近现场答辩发言，不使用过强的模板化排比句。
